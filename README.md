## Download
Run
```
git clone https://github.com/Neptune3013/Cherry-Blossom-Kitty-Fastfetch.git
```
## Installation

### Kitty Terminal theme config

Copy ``` Cherry_Blossom.conf``` or ``` Cherry_Blossom_seastone.conf ``` into ```~/.config/kitty/``` 

(if the directory does not exist, open kitty terminal and presee ctrl+shift+F2, it will create the dir and the kitty.conf file)

add 
``` 
include ./Cherry_Blossom.conf
```
OR
```
include ./Cherry_Blossom_seastone.conf
```
to ```kitty.conf``` in ```~/.config/kitty/``` depending on which you want to apply.

### Fastfetch config

copy ```config.jsonc``` into ```~/.config/fastfetch/```
